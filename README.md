# Notes App

    Uma aplicação web que permite que usuários se cadastrem, façam login e criem e editem notas de texto.

## Tecnologias utilizadas:

<br>

- Typescript
- Node
- Express
- TypeOrm
- React
- Axios
- Material UI
- Yup
- PostgreSql
- Docker
